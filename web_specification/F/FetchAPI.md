# Fetch API

Fetch API 提供了一个获取资源的接口（跨与请求）。任何使用 XMLHttpRequest 的人都能轻松的用上，而且新的 API 提供了更加强大的和灵活的功能集。

## 概念

- Request
- Response
- 终止

## Fetch 接口

- WindowOrWorkerGlobalScope.fetch()
- Headers
- Request
- Response
- Body
