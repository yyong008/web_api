# CSSOM

他是一个重要的模型：CSS 对象模型 CSSObjecModal

CSS Object Modal 是一组允许用 JavaScript 操作 CSS 的 API。他是继 DOM 和 HTML API 之后，又一个操纵 CSS 的接口，从而能动态的读取和修改 CSS 样式。
