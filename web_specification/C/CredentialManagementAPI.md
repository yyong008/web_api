# CredentialManagementAPI

此 API 允许网站存储和检索用户，联合账户和公钥证书。这个些功能允许用户在不输入密码的情况进行登录，检查他们曾经登录到一个站点的联合账户，并且在会话过期且在会话过期没有显示的登录流程的情况下回复会话。

## 用法

- Credentail
- CredentailsContainer
- FederatedCredentail
- PasswordCredentail
- PublicKeyCredential
