# Performance API

高时间采样率定义了 Performance 接口，该接口支持应用程序中客户端的演示测量

Performance 接口被任务是高采样率的，因为其精确度可达千分之一毫秒。

- DOMHighResTimeStamp
- Performance
- time
- PerformanceEntry
- PerformanceFrameTiming
- PerformanceMark
- PerformanceMeasure
- PerformanceNavigationTiming
- PerformanceObserver
- PerformanceSourceTiming
